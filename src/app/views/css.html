Basics

CSS Design Patterns

  • BEM - Block Element Modifier: .block__element_modifier

Block Construction Order

  1. html / folders
  2. bg-reference / base
  3. grid (fr > px > %)
  4. flexbox / positioning
  5. img
  6. font-* (family/size/weight/style)
  7. padding (-left? button?) (px)
  8. margin-top / margin-left (px)
  9. width / height (% > px)
  10. color / background-color

Rules Order

  1. Structure: grid, flex, position, display, and float
  2. Box model: width, height, margin, padding, and border
  3. Background
  4. Typography
  5. Other

Box Model

  • margin > border > padding > content

Selectors

  • * // universal selector (all elements)
  • div // tag selector (all <div> elements)
  • #headline // id
  • .blue // class blue
  • .blue.red // class blue and red
  • :hover // mouseover
  • ::after // after any element. Use as a complement of it.
  • ::before // before any element. Use as a complement of it.
  • div > p // child p of div
  • [attr='val'] // attribute attr equal val
  • :first-child
  • :last-child
  • :nth-child(2) // number, odd, or even
  • :nth-of-type(2) // number, odd, or even

Length units

  • % percentage
  • em relative to parent's font-size
  • rem relative to root's font-size (1rem == browser's font-size)
  • vw viewport width
  • vh viewport height
  • px pixels
  • fr fraction. Unit used for grid

Adding CSS

  • External stylesheet < link rel="stylesheet" href="">
  • Internal stylesheet < style>* { }< /style>
  • Import in CSS: @import { url(./folder/file.css); }

Media queries

  • @media (min-width: 480px) {} // Tablet
  • @media (min-width: 992px) {} // Desktop
  • @media screen and (min-width: 769px) and (max-width: 992px) {} // Exceptions
  • All @media

Comments

  • /* Comment */
  • // Comment

Lines

  • div { display: hidden; } // one rule, one line
  • div { position: absolute; top: 0; } // over one rule, multiple lines

Custom properties

  • :root { --blue: #0A0A0A; }
  • div { color: var(--blue); }

Functions

  • width: calc(50% - 100px);
  • color: var(--blue); // -- blue set in :root {}
  • radial-gradient(red, orange, yellow); // background
  • linear-gradient(0deg, red, yellow 50%, blue); // background

!important

  • color: red !important; // overrides blue
  • color: blue;

Structure

Grid

Parent

  • display: grid;
  • grid-template-rows: 100px 200px 100px;
  • grid-template-columns: 1fr repeat(10, 99px) 1fr;
  • grid: grid-template-rows / grid-template-columns;
  • gap: 10px 20px;
  • place-items: center center;

Children

  • grid-row: 1 / 3;
  • grid-column: 2 / 3;
  • place-self: end end;
  • CSS Tricks's Complete Guide to Grid

Flexbox

Parent

  • display: flex;
  • flex-direction: row | column
  • flex-wrap: nowrap || wrap; // nowrap disregard width.
  • flex-flow: flex-direction + flex-wrap
  • justify-content: x axis
  • align-content:; y axis
  • align-items:; // y axis

Children

  • flex-grow: (number)
  • flex-shrink: (number)
  • flex-basis: auto
  • flex: flex-grow + flex-shrink + flex-basis
  • order: (number)
  • align-self: Check Dev Tools's options
  • CSS Tricks's Complete Guide to Flexbox

Positioning

Parent

  • position: relative

Children

  • position: absolute
  • top: (number);
  • right: (number);
  • bottom: (number);
  • left: (number);
  • z-index: (number);
    • position: fixed // fixed in the screen despite scrolling

    Display

    • display: inline; // siblings inline are in the same line
    • display: block; // width 100%
    • display: none; // hides element

    Float

    • float: left || right;

    Centering

    • Horizontal + vertical using Grid // parent & child
    • Horizontal + vertical using Flexbox // parent & child
    • Horizontal + vertical using Position: absolute // parent & child
    • Horizontal using margin: 0 auto // no parent

    Margin

    • margin: top right bottom left; // clockwise
    • margin: 10px 100px 5px; // == 10px 100px 5px 100px
    • margin: 20px 200px; // == 20px 200px 20px 200px

    Padding

    • padding: top right bottom left; // clockwise
    • padding: 10px 100px 5px; // == 10px 100px 5px 100px
    • padding: 20px 200px; // == 20px 200px 20px 200px

    • border-left: 1px solid black;
